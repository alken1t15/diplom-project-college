Добрый день, уважаемая коммисия, мы хотим представить вам наш проект АИС экзитус. Целью нашего проекта было разработать автоматизированную информационную сиситему учебного процесса для колледжа например как классрум с расширенным функционалом. Задачи которые мы поставили перед собой для реализации проекта: 1) Нам нужно было изучить какие продукты уже существуют на рынке и взять хорошие идеи с них. 2)Используя современные технологии реализовать проект который будет быстро и стабильно работать 3) Разработать удобный интерфейс для пользователей 4) Обеспечить максимальную эффективность на нашей платформе. Идея проекта является актуальным потому что: на рынке не существует проект который подошел бы идеально для нашего учебного заведения. При разработке, мы провели анализ на востребовательность функций в приложение и выявели ряд функций которые должны присутствовать в приложении. При разработке проекта для фронтенд было использовано: : React JS, TypeScript, SCSS для бэкенд:  
Java Spring, PostgreSQL.

Чтобы зайти на сайт, нам нужно пройти авторизацию. Для авторизации используется логин и пароль, также доступна возможность смена языка.

Мы вошли в сайт от аккаунта студента. Эта главная страница студента, по левую часть экрана находится навигация, также по левую строну у нас ходится блок с личной информацией и оценки которые были получены за сегодня. По середине у нас распологается успеваемость студента по месяцам. Например мы можем выбрать сентябрь, у нас выйдет успеваемость за сентябрь. По правую часть экрана у нас находится расписание на текущий день и также есть возможность прикерепление справок об отсутствии на занятии от студента.

Далее переходим на страницу с оценками. Здесь отображаются текущие оценки студента, они разбиты по курсу, семестру, месяцам. В конце каждого месяца отображается итог за месяц.Справой староны находится список преподователей которые введут предметы.
Есть отдельная графа итоговые оценки. Здесь отображается оценки за курс и по семестрам.

Переходим на страницу с файлами. Здесь отображаются файлы, которые прекрепили преподователи к определенному курсу, у студента есть возможность скачать файлы.

Следующая страница отвечает за домашние задания студента. Влевой части мы можем выбрать определенное задание, по середине будет отображаться информация о самом задание, справой части будет возможность прекрепить само задание.

Давайте теперь зайдем в аккаунт от преподователя.

Влевой части как у студента, отображается личная информация, снизу находится задания которые выполнели студенты, по середине находится расписание преподователя, вправой части преподователь может выставлять пропуски и также смотреть справки которые прекрепил студент. 

Следующая страница, это домашние задания которые создал преподователь. Можно выбрать конкретное задание и посмотреть, у каких студентов выстовлены оценки и также посмотреть студенты которые сдали задания, можно скачать файл и посмотреть его содержимое.

Попимо выставление оценок за задание, преподователь может вручную выстовить оценку за определенный день, например за устное задание на уроке, для этого ему нужно выбрать группу и предмет по которому он хочет вытсовить оценку. Далее может выстовить оценку студенту.

На последний странице преподователь может прекрепить задание на определенный курс для всей группы. Для этого ему нужно выбрать группу, написать краткое описание файла и прекрепить сам файл. Справой части преподователь может создать новое задание, он выбирает группу, предмет, срок сдачи, название работы и описание. После этого по середени он может выбрать задание и прекрепить файл для этого задания.
